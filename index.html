Команда

Текущий режим: подготовка проекта до появления команды.

  • Руководитель/архитектор проекта: Алексей.
  • Backend: FastAPI, SQLAlchemy, Alembic, PostgreSQL.
  • Frontend: React/Vite, русская локализация, готовность к мультиязычности.
  • DevOps: Docker, GitHub Actions.
  • Миграция: извлечение данных из SPI / Oracle и загрузка в PostgreSQL.

Для руководства

  • Цель: замена SmartPlant Instrumentation с сохранением инженерных данных КИПиА.
  • Демо должно показывать карточки приборов, спецификации, контуры и навигацию по объектам.
  • SPI после миграции остаётся read-only.
  • Чертежи не должны превращаться в ручной PDF-процесс.
  • Публичный dashboard можно использовать как витрину статуса, без раскрытия исходников.
